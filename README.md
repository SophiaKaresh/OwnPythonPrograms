# OwnPythonPrograms
Krátké programy - sloužící k procvičování Pythonu

1. Minigame Guess Number - jedná se o krátkou hru, ve které uživatel hádá číslo vygenerované náhodně počítačem (od 1-50). Dohromady má uživatel 10 pokusů.
2. Day of week - pomáhá uživateli zjistit dle zadaného datumu o jaký den se jedná
